¡ñ Tricky CSS:
1. How to center text: give a width to the item, then set margin: auto.
2. How to have left and right sections: give width to left and right, which should add up to be the same as parent element's width (be careful to padding/border/margin), then set float: left/right.
3. Email should goes to href="mailto:[your email here]".
